Important FAQ:

What is theDocument Object Model(DOM)?

DOM is like tree of objects in website and using DOM we can manipulate content, styles and update dynamically.

How do you select an Element from DOM?

We can select elements from DOM using different ways, like getElementById,TagName,ClassName also querySelector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is use as event listener that attached in parent element and dynamically handle its child elements.

How do you manipulate an element's attributes, and styles using the DoM?

We can manipulate element by using setAttribute method or set up new id dynamically. also we can style dynamically by using add classList or using style property value using dot method.
